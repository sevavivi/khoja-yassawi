<h2>Introduction</h2>
<p>
This project explores how Semantic Web technologies can be used to represent and enrich cultural heritage information related to the Mausoleum of Khoja Ahmed Yasawi, one of the most important Islamic monuments in Central Asia.
</p>
<p>
Using Wikidata, RDF triples, SPARQL queries, and Large Language Models, we investigated existing knowledge gaps and proposed new semantic descriptions that better represent the religious, historical, and cultural significance of the monument.
</p>
<h2>Research Objectives</h2>
<ul>
<li>Analyse existing Wikidata information.</li>
<li>Identify semantic gaps.</li>
<li>Create RDF triples for enrichment.</li>
<li>Use SPARQL for data exploration.</li>
<li>Evaluate LLM support in Knowledge Engineering.</li>
</ul>
