Mausoleum of Khoja Ahmet Yassawi

RDF, SPARQL and Knowledge Graph Analysis of Islamic Cultural Heritage

Gulnara Kozmet & Sevara Abdulazizova

Explore Project View GitHub

Topic

This project focuses on the Mausoleum of Khoja Ahmet Yassawi, a UNESCO World Heritage Site located in Turkistan, Kazakhstan.

The monument is significant not only as an example of Timurid architecture, but also as a religious, spiritual and cultural site connected with Sufism and the Yasawiyya tradition.

Research Question

How can Wikidata for the Mausoleum of Khoja Ahmet Yassawi be enriched through RDF triples and knowledge graph modeling?

Methodology

Our methodology combined knowledge graph exploration, SPARQL querying, LLM prompting and human evaluation.

1. Topic selection
We selected the mausoleum as a cultural heritage entity represented in Wikidata.
2. SPARQL exploration
We queried Wikidata to understand the current structured description.
3. Gap identification
We identified missing semantic relations.
4. RDF enrichment
We proposed RDF triples to model missing knowledge.

SPARQL Exploration

SPARQL was used to inspect what Wikidata already contains about the mausoleum and to identify possible missing relations.

Query 1 — Existing Properties

SELECT DISTINCT ?propertyLabel ?valueLabel
WHERE {
  wd:Q46069 ?prop ?value .
  ?property wikibase:directClaim ?prop .

  SERVICE wikibase:label {
    bd:serviceParam wikibase:language "en" .
  }
}
ORDER BY ?propertyLabel
LIMIT 200

Query 2 — Yasawiyya Check

SELECT ?propertyLabel ?valueLabel
WHERE {
  wd:Q46069 ?prop ?value .
  ?property wikibase:directClaim ?prop .

  FILTER(CONTAINS(LCASE(STR(?valueLabel)), "yas"))

  SERVICE wikibase:label {
    bd:serviceParam wikibase:language "en" .
  }
}

Query 3 — Pilgrimage Check

SELECT ?propertyLabel ?valueLabel
WHERE {
  wd:Q46069 ?prop ?value .
  ?property wikibase:directClaim ?prop .

  FILTER(CONTAINS(LCASE(STR(?valueLabel)), "pilgrim"))

  SERVICE wikibase:label {
    bd:serviceParam wikibase:language "en" .
  }
}

Identified Knowledge Gaps

Wikidata already contains historical, geographical and architectural information. However, several cultural and religious aspects are underrepresented.

Yasawiyya Sufi Order
No explicit semantic connection with the Yasawiyya tradition was found.
Pilgrimage Site
The role of the mausoleum as a pilgrimage destination is not clearly represented.
Spiritual Significance
The religious and symbolic meaning of the site is not fully modeled.
Related Heritage Sites
Links with related religious heritage sites in Kazakhstan are limited.

LLM Evaluation

We compared ChatGPT and Gemini using three prompting techniques: Zero-shot, Few-shot and Chain-of-Thought prompting.

Prompt Type ChatGPT Gemini Human Assessment
Zero-shot Focused and concise RDF triples. More detailed contextual explanations. Both models were relevant.
Few-shot Followed the RDF example clearly. Adapted strongly to the example. Few-shot improved structure.
Chain-of-Thought Most aligned with the project gaps. More exploratory and broader. ChatGPT was more consistent for this project.

Proposed RDF Triples

Based on SPARQL exploration and LLM-assisted analysis, we propose the following RDF-style triples. 

:Mausoleum_of_Khoja_Ahmet_Yassawi :associatedWith :Yasawiyya_Sufi_Order .
:Mausoleum_of_Khoja_Ahmet_Yassawi :hasFunction :Pilgrimage_Site .
:Mausoleum_of_Khoja_Ahmet_Yassawi :associatedWith :Sufism .
:Mausoleum_of_Khoja_Ahmet_Yassawi :relatedHeritageSite :Arystan_Bab_Mausoleum .

Challenges

SPARQL Syntax
Some initial queries failed because of syntax problems.
No Matching Records
Some queries returned no results, which helped us identify gaps.
LLM Verification
LLM outputs required human validation.
Cultural Modeling
Religious and spiritual meaning is difficult to express through simple triples.

Conclusion

This project demonstrated how RDF, SPARQL and Large Language Models can support knowledge graph enrichment for cultural heritage.

Wikidata already contains basic information about the Mausoleum of Khoja Ahmet Yassawi, but its religious, spiritual and pilgrimage-related significance is less visible as structured data.

The proposed RDF triples help represent the mausoleum not only as an architectural object, but also as a religious and cultural heritage site.
